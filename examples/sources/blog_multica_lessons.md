# Blog Note - Multica Lessons for Agent Boards

Issue-driven agent collaboration works because the work carrier is visible.
Agents can be assigned, task runs can be inspected, and failures have an owner
and a timeline.

For Ariadne, the lesson is not to clone a full managed-agent platform. The
lesson is to make the Build Board show source, status, agent run attempts,
execution backend, changed files, tests, review verdict, and memory write-back.

## Suggested build action

Improve Build Board and Agent Run timeline to show execution status and
attempts.
