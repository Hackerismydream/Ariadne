# Codex Handoff

Use this skill when Ariadne routes a Build Ticket to a coding backend.

Rules:

- Treat the Ariadne handoff as the source of scope.
- Edit only the allowed paths named by the Build Packet.
- Do not commit, push, merge, or open a PR.
- Capture stdout, stderr, exit code, changed files, diff, and tests.
- Stop when acceptance criteria pass.
