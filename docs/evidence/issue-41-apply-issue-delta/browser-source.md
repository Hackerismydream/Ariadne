# Issue 41 Browser Source

The current Project Version must turn a generated Issue Delta into the mainline BuildTicket issue set.

The applied tickets must remain scoped to the selected target project and target version. Historical tickets from the same target repository must not appear on the current Issues board unless they belong to the selected Project Version.

Each applied issue should carry source evidence references, acceptance criteria, affected modules, build context, target project identity, target project path, and target version metadata so the Workbench issue detail can explain why the issue exists.
