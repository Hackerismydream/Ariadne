# Blog Note: Lessons from Multica-like Agent Collaboration

A multi-agent development system should not hide work in chat logs. It should use visible work carriers, assignment, run history, artifacts, and board state.

## Project implication for Ariadne

The board should show attempts, backend used, exit code, changed files, test result, reviewer verdict, and memory status.

## Suggested build action

Enhance the Build Board and Agent Run timeline.
