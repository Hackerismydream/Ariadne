# Build Skill Template

Path:

```text
.skills/<skill-name>/SKILL.md
```

Template:

```markdown
# <Skill Name>

## Description

What this skill helps the agent do.

## When to use

- Situation 1
- Situation 2

## Instructions

1. Step one.
2. Step two.
3. Step three.

## Safety

- Do not leak secrets.
- Do not modify files outside allowed paths.
- Do not commit, push, merge, or create PRs.

## Expected artifacts

- Handoff
- Execution result
- Review report
```
